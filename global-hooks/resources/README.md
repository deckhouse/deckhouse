# Вебхуки ресурсов

Вебхуки в поддиректории `resources` отвечают за управление и оптимизацию ресурсов в кластере. Они в основном выполняют задачи по настройке и корректировке конфигураций существующих Kubernetes-ресурсов для обеспечения их стабильной и эффективной работы.

## Список вебхуков

- вебхук 1
    - название: cluster_autoscaler.go
    - назначение: Удаляет лимиты ресурсов (resources.limits) из Deployment'а `cluster-autoscaler`, чтобы предотвратить его троттлинг.
    - пример использования: Автоматически запускается при обнаружении Deployment'а `cluster-autoscaler` и модифицирует его спецификацию, убирая секцию `resources.limits`.

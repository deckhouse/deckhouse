---
title: "The operator-argo module"
description: "operator-argo is a module of the Deckhouse Kubernetes Platform for deploying ArgoCD installations."
d8Edition: ee
---

This module deploys the [ArgoCD Operator](https://argocd-operator.readthedocs.io/) in the Kubernetes cluster. Using the ArgoCD Operator you can install ArgoCD in the cluster by deploying an ArgoCD Custom Resource.

Examples of using the module are provided in the section [Usage Examples](examples.html).

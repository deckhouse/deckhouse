Модуль operator-argo разворачивает [ArgoCD Operator](https://argocd-operator.readthedocs.io/) в кластере Deckhouse Kubernetes Platform. Это позволяет установить ArgoCD в кластере Deckhouse Kubernetes Platform, используя ресурс [ArgoCD](https://argocd-operator.readthedocs.io/en/latest/reference/argocd/).

Примеры использования модуля приведены в разделе [Примеры использования](examples.html).

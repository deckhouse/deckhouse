---
title: "Cloud provider — Basis Dynamix"
description: "Управление виртуальными серверами и контейнерами в Deckhouse Kubernetes Platform с помощью Basis Dynamix."
---

Взаимодействие с облачными ресурсами провайдера на базе Basis Dynamix осуществляется с помощью модуля `cloud-provider-dynamix`. Он предоставляет возможность модулю [управления узлами](../../modules/node-manager/) использовать ресурсы Basis Dynamix при заказе узлов для описанной [группы узлов](../../modules/node-manager/cr.html#nodegroup).

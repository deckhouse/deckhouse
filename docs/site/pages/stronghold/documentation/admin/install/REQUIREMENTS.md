## Preparing the infrastructure

- Minimum requirements: 8 CPUs, 16GB RAM, 60GB disk space on fast disk (400+ IOPS)
- Linux kernel version >= 5.7

## Supported OS

| Linux distribution          | Supported versions              |
| --------------------------- | ------------------------------- |
| CentOS                      | 7, 8, 9                         |
| Debian                      | 10, 11, 12                      |
| Ubuntu                      | 20.04, 22.04, 24.04      |

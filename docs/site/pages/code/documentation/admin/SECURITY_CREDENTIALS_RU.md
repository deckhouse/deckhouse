Раздел идентификаторов безопасности предоставляет полный обзор всех токенов (персональные, групповые, проектные) и ключей (SSH, GPG) на платформе Code.
Администраторы могут переключать вкладки с различными видами идентификаторов, применять фильтры и опции сортировки, и удалять или отзывать идентификаторы, чтобы ограничить доступ пользователей и повысить безопасность.

## Цели

Раздел идентификаторов безопасности используется в следующих целях:

- Предоставление полной информации обо всех идентификаторах безопасности (владелец, область использования, даты создания/истечения срока/отзыва)
- Фильтрация и сортировка идентификаторов безопасности
- Управление жизненным циклом идентификаторов: отзыв, удаление

## Доступ к идентификаторам безопасности через UI

Чтобы получить доступ к идентификаторам безопасности, перейдите в режим администратора и в боковом меню выберите пункт «Идентификаторы безопасности».
Откроется таблица идентификаторов безопасности. Раздел состоит из 4 вкладок:

- **Персональные токены доступа**: Список токенов, принадлежащих реальным пользователям
- **Ключи SSH**: Список ключей SSH, принадлежащих реальным пользователям.
- **Ключи GPG**: Список ключей GPG, принадлежащих реальным пользователям.
- **Токены доступа группы/проекта**: Список токенов, принадлежащих группам верхнего уровня или проектам.

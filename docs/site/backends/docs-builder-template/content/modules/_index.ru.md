<p class="tile__descr">Библиотека модулей доступных для использования в Deckhouse.</p>

<p>Используйте фильтр по категориям слева для поиска модулей по их функциональности.</p>


Пример:

```mermaid
graph TD
    A[Туда] --> B[Сюда]
    A --> C[Не определился]
    B --> A
    B --> C
    C --> A
    C --> B
```

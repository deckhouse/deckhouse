В этом разделе описывается процесс настройки взаимодействия Deckhouse Kubernetes Platform с внутренними и внешними хранилищами образов контейнеров.

В этом разделе описаны процессы настройки взаимодействия DKP с хранилищем образов контейнеров в уже работающем кластере. Информация о настройке хранилища образов при установке кластера приведена в разделе [«Установка платформы»](../../../installing/).

В разделе [«Управление настройками взаимодействия с хранилищем образов в кластере, полностью управляемом DKP»](managing-interaction.html) описаны процессы настройки DKP c хранилищем образов в Managed Kubernetes кластере: режимы взаимодействия и переключение между ними (подробное описание режимов — в разделе [«Архитектура режимов взаимодействия с хранилищем образов»](../../../architecture/deckhouse/registry-modes.html)).

В разделе [«Переключение работающего Managed Kubernetes кластера на использование стороннего хранилища образов контейнеров»](../registry/third-party.html) рассматривается переключение работающего Managed Kubernetes кластера (в таком кластере недоступно управление настройками взаимодействия с хранилищем образов с помощью модуля `registry`) на использование стороннего хранилища образов.

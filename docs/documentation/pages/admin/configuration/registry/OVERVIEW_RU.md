В этом разделе описывается работа с хранилищами образов контейнеров и редакциями в Deckhouse Kubernetes Platform.

## Работа с хранилищем образом контейнеров

В разделе рассматривается работа с хранилищем образов контейнеров в функционирующем кластере. Если вас интересует информация про работу с хранилищем образов при установке кластера, перейдите в раздел [«Установка платформы»](../../../installing/).

В разделе [«Стороннее хранилище образов контейнеров»](../registry/third-party.html) описан процесс переключения работающего кластера DKP на использование стороннего хранилища образов.
В разделе [«Внутреннее хранилище образов контейнеров»](../registry/internal.html) рассматривается подготовка к переключению между режимами работы кластера: с использованием внутреннего хранилища образов или без использования, а также процессы переключения.
В разделе [«Восстановление подключения к хранилищу образов»](../registry/restore-token.html) описан процесс восстановления загрузки образов Deckhouse Kubernetes Platform при истекшем лицензионном токене.

## Работа с редакциями

В разделе [«Переключение редакций»](../registry/switching-editions.html) описаны возможные варианты переключения редакций в работающем кластере Deckhouse Kubernetes Platform.

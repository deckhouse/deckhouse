---
title: "Обзор"
description: "Использование managed-services в Deckhouse Kubernetes Platform"
---

Managed-сервисы позволяют пользователям создавать и использовать готовые прикладные сервисы с помощью кастомных ресурсов Kubernetes. Обычно в пользовательских материалах описываются базовые сценарии: создание сервиса, настройка параметров экземпляра, подключение, резервное копирование, восстановление и проверка состояния.

Используйте этот раздел, чтобы:

- создать managed-сервис в своём неймспейсе;
- настроить параметры экземпляра сервиса;
- подключиться к созданному сервису;
- выполнить базовые операции сопровождения;
- проверить состояние ресурса и результат настройки.

Сведения о подготовке инфраструктуры, настройке классов сервисов и других административных параметрах приведены в разделе [«Администрирование»](../../../admin/managed-services/overview/).

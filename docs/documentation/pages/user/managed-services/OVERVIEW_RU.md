Managed-сервисы позволяют пользователям создавать и использовать готовые прикладные сервисы с помощью кастомных ресурсов Kubernetes.

Используйте этот раздел, чтобы:

- создать managed-сервис в своём неймспейсе;
- настроить параметры экземпляра сервиса;
- подключиться к созданному сервису;
- выполнить базовые операции сопровождения;
- проверить состояние ресурса и результат настройки.

Использование сервиса PostgreSQL описано в разделе [managed PostgreSQL](postgres.html).

Сведения о подготовке инфраструктуры, настройке классов сервисов и других административных параметрах приведены в разделе [«Администрирование»](../../admin/configuration/managed-services/).

Приветствуем вас на главной странице документации Deckhouse Platform Certified Security Edition — первой платформы контейнеризации, сертифицированной ФСТЭК России.

Документация содержит следующую информацию:

- [Руководство по обновлению](update.html) — процесс обновления DKP CSE с версии v1.67.4 до v1.73.1.
- [Руководство администратора](admin-guide.html) — установка, настройка и эксплуатация платформы, управление кластером, хранилищами, сетью, безопасностью и виртуализацией.
- [Руководство пользователя](user-guide.html) — работа с платформой для пользователей: развёртывание приложений, управление ресурсами и повседневные операции.
- Дополнительную информацию по подсистемам и модулям платформы.
- [Справочную информацию](/reference/api/global.html).
- [Описание утилиты Deckhouse CLI](cli/d8/) — общее описание CLI-утилиты `d8` (Deckhouse CLI) и справка по доступным командам.

Как быстро найти то, что нужно:

- Для поиска по области применения воспользуйтесь меню.

Deckhouse CLI (`d8`) — интерфейс командной строки для работы с DKP CSE. Он объединяет команды для работы с Kubernetes, администрирования Deckhouse Kubernetes Platform, доставки приложений, копирования образов, резервного копирования, сбора диагностической информации, виртуализации, работы с пользовательскими данными и других задач.

`d8` входит в поставку DKP CSE, а также автоматически устанавливается на все узлы кластера.

## Основные группы команд

Команды `d8` сгруппированы по назначению:

| Команды | Назначение                                                                                                                                                        |
|---|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [`d8 k`](../d8/reference/#d8-k) | Команды для управления кластером (аналогично `kubectl`).                                                                                                          |
| [`d8 system`](../d8/reference/#d8-system) | Команды для работы с системными настройками платформы, её конфигурацией, модулями, очередями, пакетами и системными логами.                                       |
| [`d8 status`](../d8/reference/#d8-status) | Команда для получения текущего состояния платформы.                                                                                                               |
| [`d8 backup`](../d8/reference/#d8-backup) | Команды для создания резервных копий ключевых данных кластера и диагностических выгрузок.                                                                         |
| [`d8 mirror`](../d8/reference/#d8-mirror) | Команды для копирования образов компонентов платформы в локальную файловую систему или стороннее хранилище образов контейнеров.                                   |
| [`d8 delivery-kit`](../d8/reference/#d8-delivery-kit) | Команды для сборки, публикации, планирования и развёртывания приложений, а также для работы с Helm, хранилищем образов контейнеров, SBOM и связанными операциями. |
| [`d8 data`](../d8/reference/#d8-data) | Команды для управления данными, а также их экспорта и импорта.                                                                                                    |
| [`d8 v`](../d8/reference/#d8-v) | Команды для работы с виртуальными машинами, созданными в кластере.                                                                                                |
| [`d8 stronghold`](../d8/reference/#d8-stronghold) | Команды для управления жизненным циклом секретов (модуль stronghold).                                                                                             |
| [`d8 completion`](../d8/reference/#d8-completion) | Команды для генерации shell completion.                                                                                                                           |
| [`d8 tools`](../d8/reference/#d8-tools) | Вспомогательные утилиты.                                                                                                                                          |
| [`d8 help`](../d8/reference/#d8-help) | Встроенная справка.                                                                                                                                               |

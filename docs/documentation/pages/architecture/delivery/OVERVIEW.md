This section describes the architecture of the Delivery subsystem in Deckhouse Kubernetes Platform (DKP).

The Delivery subsystem includes the following modules:
- [`operator-argo`](/modules/operator-argo/): Manages Argo CD installations in DKP.
- [`operator-helm`](/modules/operator-helm/): Declaratively manages Helm chart deployments.

## Добавьте свой хук

Подробнее с концептом хуков можно ознакомиться в [документации shell-operator](https://flant.github.io/shell-operator/HOOKS.html).

Хуки нужны, чтобы модуль мог динамечески работать с Kubernetes API. Например, чтобы среагировать на наличие объектов в кластере.

> **NOTE:** для модулей Deckhouse поддерживается написание хуков только на языке Python.

Примеры возможных хуков есть в репозитории, который мы склонировали на шаге 1. Нашему модулю хуки не нужны, поэтому пока удалим то, что есть в примере.

В этом разделе мы покажем как адаптировать уже существующий Helm-чарт и превратить его в полноценный модуль Deckhouse.

Для работы вам потребуются следующие утилиты:

* git
* sed
* [yq](https://github.com/mikefarah/yq)

Не забудьте установить их прежде чем приступать к выполнению инструкции.
